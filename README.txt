Автоматическая публикация обновлений блога на стене группы ВКонтакте.

== Description ==
Плагин больше не поддерживается. Вышла новая версия плагина под названием:
[Vkontakte Cross-Post](http://wordpress.org/extend/plugins/vkontakte-cross-post/ "Vkontakte Cross-Post")
VKontakte Group Wall Publisher – плагин для WordPress, позволяющий публиковать информацию о новых записях
вашего блога на стене группы ВКонтакте прямо со страницы публикации статьи. В основе лежит ВКонтакте API,
по средствам которого и производится авторизация и публикация записей на стене вашей группы.
Для обсуждения и получения дополнительной инфомации посетите наш сайт:<br />
[webstudy.com.ua](http://webstudy.com.ua/vkontakte-group-wall-publisher/ "WebStudy.com.ua - совершенствуйся с нами!")


== Installation ==
1. Скачайте архив содержащий файлы плагина с сайта плагина.<br />
2. Загрузите файлы плагина в директорию /wp-content/plugins/.<br />
3. Активируйте плагин в панели администратора.
